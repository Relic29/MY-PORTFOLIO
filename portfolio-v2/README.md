# Relic — SOC Console (v2)

A second take on your portfolio: a dark terminal/dashboard style with a fixed sidebar instead of a scrolling top-nav page. Same content as your main site, restructured as console "panels."

## File structure
```
portfolio-v2/
├── index.html
├── style.css
├── script.js
└── assets/
    ├── profile.jpg
    └── J_Relic_Of_Rhofl_Resume.pdf
```

## Preview locally
Open the folder in VS Code and use the Live Server extension (right-click `index.html` → Open with Live Server), same as your main site.

## Deploy as a separate project
Since your current Netlify account has hit its monthly deploy credit limit, deploy this one under your **new** Netlify account:

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop) (sign in with the new account first)
2. Drag the whole `portfolio-v2` folder onto the page
3. You'll get a live URL instantly — rename it later under Site settings if you want

Once both sites are live, compare them side by side and let me know which one you want to keep as your primary — I can then either retire the other or keep both (some people link a "classic" and a "terminal mode" toggle, if you ever want that later).
